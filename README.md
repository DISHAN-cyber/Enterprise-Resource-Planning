## 🧱 Tech Stack

| Layer        | Technology           | Purpose                 |
| ------------ | -------------------- | ----------------------- |
| ⚙️ Backend   | Java + Spring Boot   | Core ERP Logic          |
| 🤖 AI/ML     | Python               | Forecasting & Analytics |

## 🏗️ Architecture

🔥 **Microservices (API-First Architecture)**

* Independent services for each module
* Scalable and cloud-ready
* Easy integration with AI & mobile
* Supports CI/CD and DevOps workflows

## 🔐 Security

* Secure login & authorization
* Role-Based Access Control (RBAC)

* API protection
