# ERP Finance Mobile

A lightweight Flutter mobile client for the ERP AI service.

## Run locally

1. Make sure Flutter is installed and configured.
2. Start the AI service at `http://localhost:8000`.
3. Run the app in an Android emulator or device:
   ```powershell
   flutter run
   ```

> The app uses `10.0.2.2:8000` as the default backend host when running on an Android emulator.
